# course_report_tool 发布打包说明

## 交付方式

将整个 `course_report_tool` 文件夹压缩后发给使用者。建议保留：

- `app.py`
- `src/`
- `input_courses/`
- `templates/`
- `README.md`
- `requirements.txt`
- `requirement.txt`
- `安装依赖.bat`
- `启动工作台.bat`
- `使用说明.docx`

可不打包：

- `.git/`
- `.venv/`
- `__pycache__/`
- `output/`
- 临时上传目录 `_uploaded_raw/`

## 使用步骤

1. 解压项目文件夹。
2. 双击 `安装依赖.bat`。
3. 双击 `启动工作台.bat`。
4. 浏览器打开后，在页面中选择课程目录、上传原始成绩表、生成标准 02、生成报告。

## 命令行运行

仍然支持原有命令行方式：

```bash
python -m src.main --course 材料成型技术_2025-2026-1
python -m src.main --course 科研训练计划_2024-2025-2
```

如果使用本地虚拟环境：

```bash
.venv\Scripts\python.exe -m src.main --course 材料成型技术_2025-2026-1
```

## Streamlit 启动命令

```bash
.venv\Scripts\python.exe -m streamlit run app.py
```

## 注意事项

- 对方电脑需要安装 Python，建议版本为 3.8 到 3.10。
- 如果依赖安装失败，通常是网络或代理问题，可换用清华源或阿里源。
- 有试卷课程必须提供 03 和 04，不能用总评成绩替代试卷卷面分。
- 按权重拆分或随机扰动测试生成的 02 不是原始真实分项成绩，只适合历史数据补录、流程测试或估算；正式归档建议人工复核。
